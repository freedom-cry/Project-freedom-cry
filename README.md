# 数据单位参考
集中载荷(驾驶舱) $(G'_c)$:1.3t
传动系统重量$(G'_q)$:2.0t
*均布重量* $(q'_y)$:0.12~0.15t/m
*冲击系数 $(K)$*:1.1(1.416 m/s)
动力系数 $(\Phi)$:1.2(A5)
最大起吊重量 $(Q)$:45t
跨度 $(l)$:22.5m
梁身自重 $(q'_w)$:0.60t/m
小车重量 $(G_x)$:16t
小车轮距 :3600mm
前轮距重心 $t_1$ :1770mm
后轮距重心 $t_2$ :1830mm
小车左端距离左支点 $x$
操纵室距离左支点 $l_c$:2.8m

*注:均布重量是由运行机构（包括传动轴等）产生的*
*冲击系数用来修正轨道不平整带来冲击而导致的载荷变化*
*动力系数用来修正起吊时以及制动时的瞬间加速度所带来的载荷变化*
